Module: Wave

Renders a wave shape This module provides an API for setting up visual appearence, e.g., wave or background color
Version:
  • 1.0
Author:
  • Tiago Carvalheira
Example

Typical use of Wave APIs within a PVSio-web plugin module.

// Example module that uses Syringe.
define(function (require, exports, module) {
    "use strict";
    var device = {};
    device.syringe = new Syringe("salive", { top: 222, left: 96, height: 8, width: 38 });
    device.syringe.render();
});

Example

Typical use of Wave APIs within a PVSio-web plugin module.

// Example module that uses Syringe.
define(function (require, exports, module) {
    "use strict";
    var device = {};
    device.wave = new Wave("ecg", { top: 222, left: 96, height: 8, width: 38 }, { waveType: 'pleth', 
                                                        parent: 'prototype',
                                                        heartRate: 60,
                                                        waveColor: '#FFFFFF',
                                                        background: '#000000',
                                                        scanBarWidth: 10,
                                                        wavesPerScreen: 4.5
                                                    });
    device.wave.render();
});

    Wave(id, coords, opt)

    Constructor.
    Parameters:
    Name Type Description
    id String The ID of the display.
    coords Object The four coordinates (top, left, width, height) of the syringe, specifying the left, top corner, and the width and height of the div element containing the syringe. Default is { top: 0, left: 0, width: 80, height: 200 }.
    opt Object Options:
  • parent (String): the HTML element where the display will be appended (default is "body")
  • title (String): presented on top left corner with wave's color
  • heartRate (Integer) Heart rate in BPM. This will affect the speed of wave. Default is 60
  • waveColor (uInt). Default is #00FF00
  • backgroundColor (uInt). widget background color. Default is #000
  • waveType (String) Define the wave type that will be rendered. options are: ecg, co2, pleth, userDefined. Default is ecg
  • userDefinedWave (Array[Integer]) [optional] when waveType is userDefined the wave rendered will be the points in this array
  • filled (string): Default is none
  • none
  • up
  • down
  • fillColor (uInt). Default is wavecolor
  • wavesPerScreen (Integer) [optional] set the number of waves that should be rendered in each line of the screen at 60bps
  • scanBarWidth (Integer) [optional] set scanBarWidth, Default is 50

    (inner) createRectangle(svg, rectWidth, rectHeight, color, duration)

    This function will append a rectangle to wave svg and define a transition that make the rectangle go through the svg creating the ilusion of real time wave
    Parameters:
    Name Type Description
    svg d3 svg in which rectangle will be appended
    rectWidth (Integer) rectangle width. Default is 100
    rectHeight (Integer) rectangle height. Default is 200
    color (uInt) rectagle color. Default is #000
    duration (Integer) time in ms that rectangle will take to go through all svg width. Usually this duration is a function of heart rate and number of beats that exists on svg. Default is 1000
    Returns:
    void
