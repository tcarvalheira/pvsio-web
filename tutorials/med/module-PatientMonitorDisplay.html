Module: PatientMonitorDisplay

Renders a basic patient monitor display, including: current value, label, alarms, max/min values, tracings. This module provide APIs for changing the look and feel of the display element, such as position, font color, size, and tracings speed.
Version:
  • 1.0
Author:
  • Paolo Masci
Example

Typical use of PatientMonitorDisplay APIs within a PVSio-web plugin module.

// Example module that uses PatientMonitorDisplay.
define(function (require, exports, module) {
    "use strict";
    var device = {};
    device.spo2 = new PatientMonitorDisplay("spo2",
                        { top: 56, left: 164, height: 30, width: 100 });
    device.spo2.render(10); // the display renders 10
    device.spo2.alarm("ALARM"); // the display shows an alarm
});

Methods

PatientMonitorDisplay(id, coords, opt)

Constructor.
Parameters:
Name Type Description
id String The ID of the display.
coords Object The four coordinates (top, left, width, height) of the display, specifying the left, top corner, and the width and height of the (rectangular) display. Default is { top: 0, left: 0, width: 200, height: 80 }.
opt Object Options:
  • backgroundColor (String): background display color (default is black, "#000")
  • font (String): display font type (default is "sans-serif")
  • fontColor (String): display font color (default is white, "#fff")
  • align (String): text alignment (default is "center")
  • inverted (Bool): if true, the text has inverted colors, i.e., fontColor becomes backgroundColor, and backgroundColor becomes fontColor (default is false)
  • parent (String): the HTML element where the display will be appended (default is "body")
