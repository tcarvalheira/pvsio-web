Module: Alarm

This module will allow playing alarms, mute, unmute set volume up and down
Version:
  • 1.0.0
Author:
  • Tiago Carvalheira
Example

Usage of Alarm within a PVSio-web demo.

define(function(require, exports, module){
     "use strict";

     //Require the Alarm module
        require("widgets/med/Alarm")

     function main(){
         let Alarm = new Alarm(
             'example', // id of element
             {top: 100, left: 100, width: 100, height: 100 }, // coordinates
             {
                 parent: 'xpto',
                 muted: true,
                 audio: "alarm/alarm1.wav",
                 loop: true,
                 volume: 50 // percentage og volume
             } // options object
         )
         // render
         Alarm.render()
     }
})

Methods

isPaused()

returns if sound is paused or not
Returns:
this

pause()

pause sound

setVolume(volume)

set sound volume
Parameters:
Name Type Description
volume (float) value beetween 0 and 1. Default is 0.5

toggle()

Toggle on or off the alarm

volumeDown(value)

decrease volume by
Parameters:
Name Type Description
value (Float) amount to decrease. Default is 0.1

volumeUp(value)

increase volume by
Parameters:
Name Type Description
value (Float) amount to increase. Default is 0.1

hide()

This method will hide the widget.

isMuted()

return true if sound is or isn't muted.
Parameters:
Type Description
Returns:
Boolean

mute()

This method mutes the alarm, making every sound not audible

play()

this will play alarm when it is called. Plays in loop or only one time based on configuration
Parameters:
Type Description

render()

Render method for the Alarm widget

reveal()

This methor will show the widget

unmute()

This method unmutes the alarm making the sound audible again.
